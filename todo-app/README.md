# 📝 To-Do List App

A simple and interactive To-Do List web application built using HTML, CSS, and JavaScript.

---

## 🚀 Features
- ➕ Add new tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 💾 Data stored using localStorage (persists after refresh)
