# Weather App 🌦️

A simple weather application that shows weather details using an API.

## 🚀 Features
- Search weather by city
- Displays temperature and conditions
- Uses API to fetch live data

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Weather API
